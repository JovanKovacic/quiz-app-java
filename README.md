# 📌 Quiz App (Java)

> ⚠️ **Disclaimer**  
> This project was created primarily to understand and demonstrate application logic and object-oriented design.  
> It is intended as a learning project and serves as a foundation for implementing the same idea in other programming languages, as well as for future expansion into a web-based CRUD application with a database.

---

A console-based quiz application built in Java, designed to demonstrate core programming concepts and object-oriented design.

---

## 🧠 Purpose

This project was created as a foundation for understanding the logic behind a complete application workflow.

Since Java is the language I am most comfortable with, the goal was to first implement the full logic here, and then reuse the same idea in other programming languages such as C# and Python.

Additionally, this project serves as a base for future development of a web-based CRUD application with a database.

---

## 🚀 Features

- Loads quiz questions from a file  
- Randomizes question and answer order  
- Validates user input  
- Calculates final score  

---

## 🛠️ Concepts Used

- Object-Oriented Programming (OOP)  
- File handling  
- Collections (List, shuffle)  
- Exception handling  
- Input validation  

---

## ▶️ How to Run

1. Clone the repository  
2. Open the project in your IDE  
3. Run `Main.java`  

---

## 📂 Project Structure

- `Quiz` – main application logic  
- `Question` – represents a quiz question  
- `Answer` – represents a possible answer  
- `Main` – entry point  

---

## 🔮 Future Improvements

- Add database support (store questions and results)
